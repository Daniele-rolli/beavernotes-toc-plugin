# Table of Contents for Beaver Notes

Auto-updating Table of Contents from document headings.

## Features

- Type `/toc` to insert a table of contents block
- Automatically scans your note for headings (h1–h6)
- Shows indented hierarchy with heading prefixes
- Click any heading to jump to it
- Auto-updates as you edit
- Works completely offline, zero dependencies

## Usage

1. Type `/toc` in the editor to insert the block
2. The TOC populates from your headings
3. Click any entry to jump to that heading
4. The TOC stays in sync as you add/edit/remove headings

## Development

```bash
npm install
npm run build
npm run install:local
```
